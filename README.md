# Claude Skills Workshop

Caseproof internal workshop on authoring and using Claude Skills.

**▶ View the slides: https://caseproof.github.io/claude-skills-workshop/**

## What's inside

Single-page HTML deck covering:

- What Claude Skills are and when to use them
- Skill anatomy (`SKILL.md`, frontmatter, references)
- Writing effective `Use when...` descriptions
- Packaging `.skill` files
- Live demo: the `support-reply-writer` skill

## Download the demo skill

[`support-reply-writer.skill`](./support-reply-writer.skill) — the packaged skill file from the workshop. Upload it directly to Claude.

## Local preview

open index.html
